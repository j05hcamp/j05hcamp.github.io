RFID Ecosystem — Kiosk, Smart Cart, Hybrid Rack

Role: System design, HW dev, ecosystem integration Scope: End-to-end tool-control infra (Positive Fusion / Fusion Flex) Outcome: Live audits, dynamic BoM, access-controlled storage

Mobile RFID Scanner — Portable Fixed-Reader System

Role: Concept dev, industrial design, embedded integration Scope: Mobile fixed-reader, offline ops, FOD prevention Outcome: $2M+ savings via loss prevention & efficiency

Originally conceived in 2019 to extend Positive Fusion’s reach beyond networked zones, the Mobile RFID Scanner redefined what portable inventory control could be. Built from the ground up—starting with an 80/20 frame and laser-formed acrylic housing—the design has evolved through seven major iterations, each improving form factor, power management, and system integration. The latest version features a custom 3D-printed chassis, modular electronics bay, fanless micro-PC, and 7” touchscreen interface, supporting variable voltage (12–24V) and six-hour battery operation. The device enables rapid, offline audits, live asset tracking, and real-time tool recovery—bridging the gap between fixed infrastructure and field mobility.

TaskJet — Service Request System

Role: Product strategy, UX architecture, systems design Scope: Real-time service requests for facilities Outcome: Functional, deploy-ready platform

TaskJet was conceived during a 2025 field visit after observing inefficiencies in how managers handled constant in-person requests. The result is a lightweight, browser-based platform that replaces informal communication with structured, traceable workflows. Developed using modern visual development tools and guided by systematic design thinking, the app includes secure user authentication, configurable service types, live dashboards, and audit-ready reporting. The current build is fully operational and ready for deployment, with ongoing refinements to expand capability and scalability.

  • PIN login for kiosk/desktop
  • Configurable categories & request types
  • Real-time ticket tracking with chat
  • Role-based permissions
  • Analytics & audit history

Takeaway: TaskJet transforms constant interruptions into an organized service network—demonstrating how practical system design and iterative development can evolve real-world challenges into intelligent, production-ready tools.

Process

01
Discover & Define

Walk the floor, map the workflow, define constraints and success criteria (what “done” means in the real world).

02
Model & Prototype

Prove it small and sturdy. Validate reads, ergonomics, and serviceability. Convert to manufacturable CAD and materials.

03
Integrate & Align

Define API requirements, map UI touchpoints, and specify operator feedback signals. Coordinate with software engineers to close the loop with audits and accountability.

04
Deploy & Improve

Ship what’s durable, document what matters, train teams, capture feedback, and iterate for measurable gains.

Principles

Ownership over hand-offs.
Carry projects across the line. Reduce friction between siloed teams.
Operators are the users.
Every system is built to serve the people who use it daily. Designs should remove friction and provide immediate, intuitive feedback—whether through physical layout, interface cues, or equipment function.
Make it maintainable.
Fast to service, easy to train, and built from parts you can actually get.
